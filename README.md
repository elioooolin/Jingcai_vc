# 爱睦月子餐 - 微信小程序

专业月子餐服务管理系统，为产妇提供营养均衡、口感美味的月子餐服务。

## 项目概述

这是一个基于微信小程序开发的月子餐服务管理系统，包含客户端和管理员端功能。

### 主要功能

#### 客户功能
- 🔐 微信一键登录
- 📅 日期选择点餐
- 🍽️ 多餐次菜品选择（早餐、午餐、晚餐、高补餐）
- 📋 订单历史查看
- 👤 个人信息管理
- ❤️ 菜品收藏功能

#### 管理员功能
- 📊 订单管理和确认
- 👥 客户信息管理
- 📈 数据统计分析
- 📄 数据导出（Excel/PDF）
- ⚙️ 系统管理

### 技术栈

- **框架**: 微信小程序原生开发
- **语言**: TypeScript
- **UI组件**: TDesign 组件库
- **样式**: WXSS + CSS变量
- **状态管理**: 本地存储 + 页面状态

## 开发指南

### 环境要求

- 微信开发者工具
- Node.js >= 14
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发流程

1. 使用微信开发者工具打开项目
2. 导入项目目录 `miniprogram/`
3. 配置 AppID（可使用测试号）
4. 编译预览

### 品牌设计

项目采用"爱睦"品牌设计规范：

#### 色彩方案
- **主色调**: `#d4a574` (温暖金色)

## 功能特性

### 用户体验
- 📱 响应式设计，适配不同屏幕
- 🎨 品牌化UI设计，视觉统一
- ⚡ 快速加载，流畅交互
- 🔒 安全登录，数据保护

### 业务流程
1. **客户注册**: 管理员创建客户账号
2. **微信登录**: 客户使用微信快速登录
3. **选择日期**: 客户选择用餐日期
4. **点餐下单**: 按餐次要求选择菜品
5. **订单确认**: 管理员确认客户订单
6. **数据导出**: 管理员导出备餐数据

### 数据管理
- 客户信息管理
- 订单状态跟踪
- 营养数据分析
- 统计报表生成

## 部署说明

### 小程序发布
1. 在微信开发者工具中点击"上传"
2. 填写版本号和项目备注
3. 登录微信公众平台
4. 提交审核
5. 审核通过后发布

### 注意事项
- 确保所有页面路径配置正确
- 检查组件依赖是否完整
- 测试各功能模块正常运行
- 验证数据存储和读取逻辑
